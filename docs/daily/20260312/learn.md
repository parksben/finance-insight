### 1. 中信证券：经济数据解读框架
**来源**：[中信证券研究 / 格隆汇](https://www.gelonghui.com/p/2214426)

**核心观点**：
- 外贸韧性超预期，生产景气度维持；内需仍待提振，居民消费是下半年的核心变量
- 中美关税博弈初现破冰，二季度"抢出口"逻辑仍有效；叠加以旧换新，Q2 GDP有望维持5%+
- 消费策略：低预期+低估值+企稳趋势 → 提升资金配置偏好
  - **新消费**：品质升级、情绪经济、技术迭代催生新方向
  - **传统消费**：基本面触底，Q2有望成为压力见底窗口
- 银行投资：两条主线 —— ①高性价比（25年业绩增速高+分红优势）；②中长期配置（商业模型特色+ROE高+波动低）
- 债市：短端先行回落，DR001目标1.1%~1.4%，若短债下行则长债利率存进一步下破空间

**方法论提炼**：
> **"双主线框架"**：在关税扰动+内需修复周期下，同时追踪"出口链景气变化"和"内需消费延续性"两条主线，用以判断组合权重倾斜方向。

### 3. 达利欧最新观点：黄金、AI与全球秩序
**来源**：[聪明投资者 / 富途牛牛](https://news.futunn.com/post/69620988/)；[桥水基金观点引述 / 雪球](https://xueqiu.com/S/SH600111/time?page=92)

**核心观点**：
- "二战后以规则为基础的全球秩序正在瓦解"—— 这是达利欧近期长文的核心论点
- 黄金在全球秩序重构背景下，作为"去美元化"资产的配置价值持续上升
- 中美发展AI的体系不同：中国由国家主导，美国由市场主导；两种体系将长期竞争
- 关税对经济的实际影响：经济学家过高估计了短期通胀效应，低估了供应链重组的结构性影响

**方法论提炼**：
> **"大周期定位法"**：通过跟踪全球大国博弈的长周期信号（债务周期、储备货币变迁、地缘冲突），判断资产配置中黄金/大宗商品的战略仓位比例，而非仅看短期价格波动。

### 4. HSBC资产管理：AI驱动股市逻辑
**来源**：[汇丰银行 - 每月市场展望 2026年3月](https://www.hsbc.com.hk/zh-cn/wealth/insights/asset-class-views/investment-monthly/positive-outlook-remains-amid-tech-sell-off-and-new-tariffs/)

**方法论提炼**：
