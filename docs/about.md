# 关于澜投研

澜投研是一套基于多 Agent 协作的投资分析框架，由小小澜驱动。

## 框架设计理念

用「投资委员会」机制替代单一视角——多角色分工、对抗性验证、裁决者综合拍板。

**目标：收益最大化 + 系统性风险控制 + 方法论持续优化。**

## 角色分工

| 角色 | 职责 | 偏向 |
|------|------|------|
| 🔍 Researcher | 信息采集、市场数据、宏观事件 | 客观中立 |
| 📈 Strategist | 投资信号、建仓/平仓方案 | 进攻型 |
| 🛡️ RiskGuard | 风险审查、仓位上限、止损设计 | 保守型 |
| 🔴 Challenger | 寻找逻辑漏洞、反向论据 | 批判型 |
| ⚖️ Arbiter | 综合裁决、记录分歧 | 综合型 |

## 标准协作流程

```
Researcher → Strategist → [RiskGuard + Challenger 并行] → Arbiter
```
